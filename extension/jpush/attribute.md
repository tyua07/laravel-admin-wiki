4.2.2 设置极光推送属性
===



* 设置app_key

```
Jpush::setAppKey($app_key)
```

* 设置master_secret

```
Jpush::setMasterSecret($master_secret)
```

* 设置通知栏样式ID

```
Jpush::setBuilderId($setBuilderId)
```

* 设置通知提示声音

```
Jpush::setSound($sound)
```

* 设置应用角标

```
Jpush::setBadge($badge)
```

* 设置推送选项

```
Jpush::setOption($option)
```
